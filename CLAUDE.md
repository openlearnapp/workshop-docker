# Workshop: Docker & Docker Compose

## Purpose

IT workshop teaching container technology from the ground up — Docker commands, images, Dockerfiles, Docker Compose, and multi-container applications.

## Target Audience

- **Level**: Developers new to containers
- **Prior knowledge**: Basic command line, familiarity with running services
- **Goal**: Build, run, and orchestrate containers confidently

## Structure

- **12 lessons** from basics to multi-container apps
- **Interface languages**: Deutsch, English, Farsi, Arabic
- **Teaching language**: matches interface language (Docker is language-agnostic)
- **Features**: code blocks, commands, real-world examples

## Labels

`IT`

## Conventions

- Examples are **commands and code snippets**, not translations
- `q` = concept/question, `a` = example command or explanation
- Lessons build on each other — order matters
- Hands-on: learners should run commands themselves

## Development

```bash
# Generate audio (Edge TTS) for each interface language
bash generate-audio.sh deutsch/docker
bash generate-audio.sh english/docker
bash generate-audio.sh farsi/docker
bash generate-audio.sh arabic/docker
```
